# Add visual controls in Dynamics 365 for phones and tablets 

[!INCLUDE[cc-applies-to-update-9-0-0](../includes/cc_applies_to_update_9_0_0.md)]
 
 Use visual controls in [!INCLUDE[pn_Mobile_Express_short](../includes/pn-mobile-express-short.md)] and tablets to help mobile users enter [!INCLUDE[pn_crm_shortest](../includes/pn-crm-shortest.md)] data faster, and to provide a richer visual experience. This set of custom controls includes sliders, switches, star ratings, video embedding, and a calendar control that you can use to give users a view of their activities in a calendar format instead of a list.
