# Set up enhanced email

For sales reps to be able to use the enhanced email, you must first enable it.

To enable,

1. At the bottom of the site map, select **Change area** ![Image alt text](media/change-area-icon.png "Image hover text") , and select **App Settings**.

2. Under **Sales Administration**, select **Enhanced email**.

3. On the **Enhanced email** page, select the **Enable enhanced email** toggle to **Yes**.

4. Select **Apply**.
