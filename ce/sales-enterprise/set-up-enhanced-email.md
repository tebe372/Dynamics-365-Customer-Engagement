# Preview: Set up enhanced email

[!INCLUDE [cc-beta-prerelease-disclaimer](../includes/cc-beta-prerelease-disclaimer.md)]

Before sales reps can use enhanced email, you must enable it.

> [!IMPORTANT]
> - The enhanced email experience is only available in the Sales Hub app.
> - [!INCLUDE[cc_preview_features_definition](../includes/cc-preview-features-definition.md)]  
> - [!INCLUDE[cc_preview_features_expect_changes](../includes/cc-preview-features-expect-changes.md)]
> - [!INCLUDE[cc-preview-features-no-ms-support](../includes/cc-preview-features-no-ms-support.md)]


To enable enhanced email:

1. At the bottom of the site map, select **Change area** ![Change area](media/change-area-icon.png "Change area") and then select **App Settings**.

2. Under **Sales Administration**, select **Enhanced email**.

3. On the **Enhanced email** page, change the **Enable enhanced email** toggle to **Yes**.

4. Select **Apply**.
