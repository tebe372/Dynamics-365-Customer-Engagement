# Enable org chart (preview)

[!INCLUDE [cc-beta-prerelease-disclaimer](../includes/cc-beta-prerelease-disclaimer.md)]

Enable the org chart feature for sellers and sales managers to use to build an organization chart of the customers. Building the org chart helps sellers and sales managers to get an understanding of the account hierarchies and level of influence. 

>[!NOTE]
>- The older version org chart feature is still available for users even if the preview is not enabled. However, improved experience such as, relationship health and labels will not be available. For more details about the previous version of org chart, see [Maintain an external org chart](live-org-chart.md).
>- To go back to the previous version of org chart, disable the org char preview.
>- The org chart feature is supported for accounts and contacts.

## License and role requirements
| Requirement type | You must have |
|-----------------------|---------|
| **License** | Dynamics 365 Sales Premium, Dynamics 365 Sales Enterprise, or Dynamics 365 Sales Professional <br>More information: [Dynamics 365 Sales pricing](https://dynamics.microsoft.com/sales/pricing/) |
| **Security roles** | System Administrator <br> More information: [Predefined security roles for Sales](security-roles-for-sales.md)|

## To enable the org chart feature

1.	Sign in to your sales app, and in the lower-left corner of the page, go to **Change area** > **App Settings** > **Org chart**.
2.	On the **Org chart** page, enable the feature.

    :::image type="content" source="media/org-chart-enable-org-chart-feature.png" alt-text="Screenshot showing org chart feature enabled" lightbox="media/org-chart-enable-org-chart-feature.png"::: 

	The enhanced org chart feature is enabled in your organization and users will see the org chart with more information about the contacts and hierarchy.
