Device form factor

## Syntax 
`getFormFactor()`

## Return Value

**Type**: `number`

**Description**: Returns one of the following values

|Value| Form Factor|
|---|---|
|0|Unknown|
|1|Desktop|
|2|Tablet|
|3|Phone|
