# CaptureImageOptions

[!INCLUDE [cc_applies_to_update_9_0_0]](../../../includes/cc_applies_to_update_9_0_0.md)

## allowEdit

Whether to edit the image before saving.

**Type**:  `boolean`

## height
Height of the image to capture.

**Type**: `number`

## preferFrontCamera
Whether to capture image using the front camera of the device.

**Type**: `boolean`

## quality 
Quality of the image file in percentage.

**Type**: `number`

## width
Width of the image to capture.

**Type**: `number`
