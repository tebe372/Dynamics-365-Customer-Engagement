# refresh (Client API reference)

[!INCLUDE[](../../../../includes/cc_applies_to_update_9_0_0.md)]

Refreshes the data displayed in a timelinewall and timer control.

## Control types supported

timelinewall, timer

## Syntax

`formContext.getControl(arg).refresh();`
