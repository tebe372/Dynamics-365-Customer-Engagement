# Find and install the Outlook add-in

## Prerequisites

- Outlook client:
  - Outlook on the Web
  - Outlook 2013 or later on Windows (or Mac)
  - Outlook 2016 or later on Windows (or Mac)
  - Outlook 2019 or later on Windows (or Mac)
  - Outlook on Windows (Microsoft 365)
  - Outlook on Mac (Microsoft 365)
- Dynamics 365 Field Service account and an environment with at least one entry in each of these fields:
  - Service Account
  - System Status
  - Work Order Type
  - Primary Incident Type

## Install the Outlook add-in

1. [Install the Outlook add-in](https://support.microsoft.com/office/get-an-office-add-in-for-outlook-1ee261f9-49bf-4ba6-b3e2-2ba7bcab64c8#:~:text=1%20In%20Outlook%2C%20click%20Home%20%3E%20Store%20on,trials%2C%20select%20Get%20it%20to%20...%20See%20More).

1. In the **Add-Ins for Outlook** dialog, search for and select **Viva Frontline**.

1. Select **Add**.

## Open the Outlook add-in

Once installed, the add-in displays as a button in the ribbon. When you're reading emails or composing a new one, select the add-in button to open it.

To add it to your favorites:

1. Select **View** > **View settings**.

1. Select **Mail** > **Customize actions**.

1. Under **Message surface**, check **Viva Frontline**.

1. Select **Save**. Close settings.
