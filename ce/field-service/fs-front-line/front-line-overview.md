# Dynamics 365 Field Service (Preview)

The Dynamics 365 Field Service (Preview) provides the ability to create and view work orders from where you work: Outlook, Teams, or Viva Connections. Quickly create work orders using copilot capabilities or manually while the information is in front of you without having to log into Field Service.
