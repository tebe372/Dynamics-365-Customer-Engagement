# Copilot in Field Service

Copilot in Field Service is part of the Field Service (Preview) Outlook add-in. Use the copilot to quickly create a work order with information from an email. For more information, see [Field Service (Preview) Outlook add-in with Copilot for the Frontline Manager](front-line-outlook.md).

For more information on other features of Field Service (Preview), see [Dynamics 365 Field Service (Preview) overview](front-line-overview.md).
