# Omni-channel Engagement Hub for agent using Customer Service Hub app

[!INCLUDE[cc-applies-to-update-9-0-0](../../../includes/cc_applies_to_update_9_0_0.md)]

The Omni-channel Engagement Hub on a web browser is a solution on Dynamics 365 Customer Service Hub app. As an agent, you can manage the work items assigned to you via queues. This helps you to focus on the tasks at hand and ensure better customer service. The application empowers you to work on simultaneous conversations with multiple customers. You view the holistic data of the customer with the details of their previous interaction, which ensures you have all the available information to resolve the customer issue.
