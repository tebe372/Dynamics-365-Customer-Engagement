## Dynamics 365 for Sales Professional, April 2019 release

Dynamics 365 for Sales Professional (app version 9.1.0.0) introduces the following capabilities:

-  Simplified first run experience: A role based welcome screen to help app users discover relevant video, learning modules and text documentation about the application. [!INCLUDE[proc-more-information](../includes/proc-more-information.md)] [Overview of Dynamics 365 for Sales Professional](sales-professional-overview.md)

-  In-app application settings: All application related configurations can be made from within the Sales Professional app. Administrator do not need to switch apps or go to multiple pages to set up the app and make their sales teams productive. [!INCLUDE[proc-more-information](../includes/proc-more-information.md)] [Overview of admin settings](admin-settings-overview.md)

-  Simplified standard settings: Simplified experience to integrate with Sharepoint, Exchange, import data and add users from within the Sales Professional app. [!INCLUDE[proc-more-information](../includes/proc-more-information.md)] [Overview of admin settings](admin-settings-overview.md)
