# onsizechanged event (CIF JavaScript API reference)

The `onsizechanged` event is invoked when the side panel width is changed.

## eventData

The `onsizechanged` event takes following eventData:

| eventData | Example |
|-----------|---------|
| ```{"value": <number>}``` | ```{"value": 35"}``` |
