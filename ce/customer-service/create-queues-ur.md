# Create a queue for the chat channel

In this section, we cover information on how to create a queue for the chat channel.
In the left pane, select Queues, and then complete the following steps to create a queue for the chat channel:
1.	On the Queues page, select New.
2.	On the Create a queue page, enter a name for the queue.
3.	Select Messaging for Type and enter a value for Priority as required by your business need.
