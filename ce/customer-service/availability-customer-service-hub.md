In 2020 release wave 1, the Customer Service Hub solution that is built on the Unified Interface framework will be automatically installed in all the environments that have the web client installed.

The salient features of the integrated solution are as follows:

- Existing customers can access **Customer Service Hub** in their environment. To learn more: [Introduction to Customer Service Hub](https://docs.microsoft.com/dynamics365/customer-service/user-guide-customer-service-hub)
- Users with the **Customer service app access** security role will be able to view and use the app. To learn about granting or revoking the role, see [Manage access to apps by using security roles](https://docs.microsoft.com/en-us/dynamics365/customerengagement/on-premises/customize/manage-access-apps-security-roles)
- If you delete the app manually, it will be installed automatically during the next update cycle.
