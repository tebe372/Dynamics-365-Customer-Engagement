# Preview: focus (app profile manager)

Sets the focus on a session.

## Syntax

`Microsoft.Apm.getSession(sessionId).focus();`

## Return value

None
## Example

```JavaScript
Microsoft.Apm.getSession("session-id-4").focus();
```
