# Security roles automatically assigned to users by apps

[!INCLUDE[cc-applies-to-update-9-0-0](../includes/cc_applies_to_update_9_0_0.md)]<br/>[!INCLUDE[cc-applies-to-update-8-2-0](../includes/cc_applies_to_update_8_2_0.md)]

|     | role1        |         |

|**Dynamics 365 for Project Service Automation**    |         |         |

|     | role2        |         |
|     | role3        |         |
|**Dynamics 365 for Finance and Operation**    |         |         |

You can remove security roles for users. Follow these steps.

1. Go to **Settings** > **Security** > **Users**.
2. Select a user and then choose **Manage Roles**.
3. Deselect a security role.
4. Choose **OK**.
