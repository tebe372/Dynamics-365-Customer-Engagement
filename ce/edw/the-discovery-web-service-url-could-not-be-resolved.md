# The Discovery Web Service URL could not be resolved

## Cause 
 The Discovery Web Service URL could not be resolved.  
  
## Solution
 Make sure that the URL is in the correct format, such as *auth.crm.consolidatedmessenger.com:443*, and that the appropriate DNS entries exist.
