# Data protections

Use the **Settings** > **Email marketing** > **Data protections** page to enable data-privacy features of Dynamics 365 Marketing. These currently focus on helping organizations comply with GDPR regulations in the European Union. For more information about GDPR, including how to use the settings here, see [Data protection and GDPR](gdpr.md).
