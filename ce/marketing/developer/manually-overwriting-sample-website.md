# Manually overwriting sample website 

It happens that some clients are doing their customizations directly on the Event Management sample website, so in order not to override clients' customizations, we don't update the sample website as a part of regular Event Management update. If you want to get the sample website that matches to you current Event Management version you can manually download appropriate zip containing sample data, and import it using [The Configuration Migration tool](https://docs.microsoft.com/en-us/dynamics365/customer-engagement/admin/import-configuration-data).

To get the correct sample website version, please open your CRM instance and go to the settings > solutions section. Note which **Event Management for Dynamics 365** solution version you have and refer to the following table to download appropriate zip file containing data.
