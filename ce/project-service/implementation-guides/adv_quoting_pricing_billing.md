# Advanced quoting, pricing, and billing guide

This document explains some of the advanced features in Dynamics 365 Project Service Automation that help professional service firms define their pricing, quoting, sales process, and billing. Project Service Automation uses organizational units to help a company define a set of billable roles that have distinct cost structures, which helps you understand the costs of a project.
