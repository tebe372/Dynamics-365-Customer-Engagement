# getInitialUrl (Client API reference)



Returns the default URL that an IFRAME control is configured to display. 

## Control types supported

iframe

## Syntax

`formContext.getControl(arg).getInitialUrl();`

## Return Value

**Type**: String

**Description**: The initial URL.
