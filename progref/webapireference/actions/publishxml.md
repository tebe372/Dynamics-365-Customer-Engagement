# PublishXml Action
[!INCLUDE[./descriptions/publishxml.md](./descriptions/publishxml.md)]

The PublishXml action:
* Is not bound to any entity types.
* Does not return a value.



## Parameters 
Parameters allow for data to be passed to the action.
|Name|Type|Nullable|Unicode|Description|
|----------|-----------------|----------|-----------------|-----------------|  
|ParameterXml|Edm.String|false|false|The XML that defines which solution components to publish in this request. |
