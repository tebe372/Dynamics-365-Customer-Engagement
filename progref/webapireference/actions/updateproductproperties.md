# UpdateProductProperties Action
[!INCLUDE[./descriptions/updateproductproperties.md](./descriptions/updateproductproperties.md)]

The UpdateProductProperties action:
* Is not bound to any entity types.
* Does not return a value.



## Parameters 
Parameters allow for data to be passed to the action.
|Name|Type|Nullable|Unicode|Description|
|----------|-----------------|----------|-----------------|-----------------|  
|PropertyInstanceList|Collection([dynamicpropertyinstance EntityType](../entitytypes/dynamicpropertyinstance.md))|false|true|The property instances (dynamic property instances) for a product to be updated. |
