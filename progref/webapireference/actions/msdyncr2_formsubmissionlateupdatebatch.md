# msdyncr2_FormSubmissionLateUpdateBatch Action
[!INCLUDE[./descriptions/msdyncr2_formsubmissionlateupdatebatch.md](./descriptions/msdyncr2_formsubmissionlateupdatebatch.md)]

This action is added by the [LinkedIn Lead Gen Forms (Link to Marketing) Solution](../solutions/linkedinleadgenintegrationlink.md).

The msdyncr2_FormSubmissionLateUpdateBatch action:
* Is not bound to any entity types.
* Does not return a value.
