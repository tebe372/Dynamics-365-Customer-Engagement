# UpdateFeatureConfig Action
[!INCLUDE[./descriptions/updatefeatureconfig.md](./descriptions/updatefeatureconfig.md)]

The UpdateFeatureConfig action:
* Is not bound to any entity types.
* Does not return a value.



## Parameters 
Parameters allow for data to be passed to the action.
|Name|Type|Nullable|Unicode|Description|
|----------|-----------------|----------|-----------------|-----------------|  
|FeatureType|Edm.Int32|false|true|internal|
|ConfigData|Edm.String|false|false|internal|
