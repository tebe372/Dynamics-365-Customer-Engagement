# msdyn_IoTSendTestAlert Action
[!INCLUDE[./descriptions/msdyn_iotsendtestalert.md](./descriptions/msdyn_iotsendtestalert.md)]

The msdyn_IoTSendTestAlert action:
* Is not bound to any entity types.
* Does not return a value.



## Parameters 
Parameters allow for data to be passed to the action.
|Name|Type|Nullable|Unicode|Description|
|----------|-----------------|----------|-----------------|-----------------|  
|IoTDeviceIds|Edm.String|false|false|[Reserved for future use.] |
