# msdyn_applyworktemplateforresources Action
[!INCLUDE[./descriptions/msdyn_applyworktemplateforresources.md](./descriptions/msdyn_applyworktemplateforresources.md)]

The msdyn_applyworktemplateforresources action:
* Is not bound to any entity types.
* Does not return a value.



## Parameters 
Parameters allow for data to be passed to the action.
|Name|Type|Nullable|Unicode|Description|
|----------|-----------------|----------|-----------------|-----------------|  
|calendarIds|Edm.String|false|false||
|templateCalendarId|Edm.String|true|false||
