# adx_SendEmailConfirmationToContact Action
[!INCLUDE[./descriptions/adx_sendemailconfirmationtocontact.md](./descriptions/adx_sendemailconfirmationtocontact.md)]

This action is added by the [Dynamics 365 Portals - Identity Workflows Solution](../solutions/microsoftidentityworkflows.md).

The adx_SendEmailConfirmationToContact action does not return a value.
