# UnlockSalesOrderPricing Action
[!INCLUDE[./descriptions/unlocksalesorderpricing.md](./descriptions/unlocksalesorderpricing.md)]

The UnlockSalesOrderPricing action:
* Is not bound to any entity types.
* Does not return a value.



## Parameters 
Parameters allow for data to be passed to the action.
|Name|Type|Nullable|Unicode|Description|
|----------|-----------------|----------|-----------------|-----------------|  
|SalesOrderId|Edm.Guid|false|true|The ID of the sales order. |
