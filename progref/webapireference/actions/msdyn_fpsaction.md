# msdyn_FpsAction Action
[!INCLUDE[./descriptions/msdyn_fpsaction.md](./descriptions/msdyn_fpsaction.md)]

The msdyn_FpsAction action is not bound to any entity types.



## Parameters 
Parameters allow for data to be passed to the action.
|Name|Type|Nullable|Unicode|Description|
|----------|-----------------|----------|-----------------|-----------------|  
|Type|Edm.Int32|false|true||
|InputParameter|Edm.String|true|false||

## Return Type
The msdyn_FpsAction action returns the following value.
|Type|Nullable|Description| 
|----------|-----------------|-----------------|    
|[msdyn_FpsActionResponse ComplexType](../complextypes/msdyn_FpsActionResponse.md)|false|msdyn_FpsActionResponse|
