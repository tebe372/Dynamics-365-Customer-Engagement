# GetTrackingTokenEmail Action
[!INCLUDE[./descriptions/gettrackingtokenemail.md](./descriptions/gettrackingtokenemail.md)]

The GetTrackingTokenEmail action is not bound to any entity types.



## Parameters 
Parameters allow for data to be passed to the action.
|Name|Type|Nullable|Unicode|Description|
|----------|-----------------|----------|-----------------|-----------------|  
|Subject|Edm.String|true|false|The context of the email.  |

## Return Type
The GetTrackingTokenEmail action returns the following value.
|Type|Nullable|Description| 
|----------|-----------------|-----------------|    
|[GetTrackingTokenEmailResponse ComplexType](../complextypes/GetTrackingTokenEmailResponse.md)|false|Contains the response from the GetTrackingTokenEmail Action. |
