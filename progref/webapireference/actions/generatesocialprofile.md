# GenerateSocialProfile Action
[!INCLUDE[./descriptions/generatesocialprofile.md](./descriptions/generatesocialprofile.md)]



## Bound Entities 
Bound actions are invoked by appending the action name to the URI representing an entity or collection.
|EntityType|Binding Type|
|----------|-----------------|
|[socialprofile Entitytype](../entitytypes/socialprofile.md)|collection|

## Parameters 
Parameters allow for data to be passed to the action.
|Name|Type|Nullable|Unicode|Description|
|----------|-----------------|----------|-----------------|-----------------|  
|Entity|[socialprofile EntityType](../entitytypes/socialprofile.md)|false|true|The social profile instance to return. |

## Return Type
The GenerateSocialProfile action returns the following value.
|Type|Nullable|Description| 
|----------|-----------------|-----------------|    
|[socialprofile EntityType](../entitytypes/socialprofile.md)|false|Contains the response from GenerateSocialProfile action.|
