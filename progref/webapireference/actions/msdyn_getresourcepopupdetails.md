# msdyn_GetResourcePopupDetails Action
[!INCLUDE[./descriptions/msdyn_getresourcepopupdetails.md](./descriptions/msdyn_getresourcepopupdetails.md)]

The msdyn_GetResourcePopupDetails action is not bound to any entity types.



## Parameters 
Parameters allow for data to be passed to the action.
|Name|Type|Nullable|Unicode|Description|
|----------|-----------------|----------|-----------------|-----------------|  
|ResourceId|Edm.String|false|false||

## Return Type
The msdyn_GetResourcePopupDetails action returns the following value.
|Type|Nullable|Description| 
|----------|-----------------|-----------------|    
|[msdyn_GetResourcePopupDetailsResponse ComplexType](../complextypes/msdyn_GetResourcePopupDetailsResponse.md)|false|msdyn_GetResourcePopupDetailsResponse|
