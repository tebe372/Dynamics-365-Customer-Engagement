# msdyn_RegisterIoTDevice Action
[!INCLUDE[./descriptions/msdyn_registeriotdevice.md](./descriptions/msdyn_registeriotdevice.md)]

The msdyn_RegisterIoTDevice action:
* Is not bound to any entity types.
* Does not return a value.



## Parameters 
Parameters allow for data to be passed to the action.
|Name|Type|Nullable|Unicode|Description|
|----------|-----------------|----------|-----------------|-----------------|  
|IoTDeviceIds|Edm.String|false|false|A comma separated list of IoT Device record GUIDs, each to be registered.|
