# CalculateActualValueOpportunity Action
[!INCLUDE[./descriptions/calculateactualvalueopportunity.md](./descriptions/calculateactualvalueopportunity.md)]

The CalculateActualValueOpportunity action does not require any parameters.

## Bound Entities 
Bound actions are invoked by appending the action name to the URI representing an entity or collection.
|EntityType|Binding Type|
|----------|-----------------|
|[opportunity Entitytype](../entitytypes/opportunity.md)|entity|



## Return Type
The CalculateActualValueOpportunity action returns the following value.
|Type|Nullable|Description| 
|----------|-----------------|-----------------|    
|[CalculateActualValueOpportunityResponse ComplexType](../complextypes/CalculateActualValueOpportunityResponse.md)|false|Contains the response from CalculateActualValueOpportunity action. |
