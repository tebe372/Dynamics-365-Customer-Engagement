# adx_AzureBlobStorageEnabled Action
[!INCLUDE[./descriptions/adx_azureblobstorageenabled.md](./descriptions/adx_azureblobstorageenabled.md)]

This action is added by the [Dynamics 365 Portals - Azure Storage Solution](../solutions/microsoftazurestorage.md).

The adx_AzureBlobStorageEnabled action:
* Is not bound to any entity types.
* Does not require any parameters.
