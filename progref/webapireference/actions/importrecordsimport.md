# ImportRecordsImport Action
[!INCLUDE[./descriptions/importrecordsimport.md](./descriptions/importrecordsimport.md)]

The ImportRecordsImport action does not require any parameters.

## Bound Entities 
Bound actions are invoked by appending the action name to the URI representing an entity or collection.
|EntityType|Binding Type|
|----------|-----------------|
|[import Entitytype](../entitytypes/import.md)|entity|



## Return Type
The ImportRecordsImport action returns the following value.
|Type|Nullable|Description| 
|----------|-----------------|-----------------|    
|[asyncoperation EntityType](../entitytypes/asyncoperation.md)|false|The asynchronous import records job.|
