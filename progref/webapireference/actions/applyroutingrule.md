# ApplyRoutingRule Action
[!INCLUDE[./descriptions/applyroutingrule.md](./descriptions/applyroutingrule.md)]

The ApplyRoutingRule action:
* Is not bound to any entity types.
* Does not return a value.



## Parameters 
Parameters allow for data to be passed to the action.
|Name|Type|Nullable|Unicode|Description|
|----------|-----------------|----------|-----------------|-----------------|  
|Target|[crmbaseentity EntityType](../entitytypes/crmbaseentity.md)|false|true|The incident to apply the routing rule to. |



## Entities
Use the ApplyRoutingRule action with this entity type:
|EntityType |Description|
