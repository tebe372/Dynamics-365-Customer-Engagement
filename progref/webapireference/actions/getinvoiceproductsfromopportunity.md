# GetInvoiceProductsFromOpportunity Action
[!INCLUDE[./descriptions/getinvoiceproductsfromopportunity.md](./descriptions/getinvoiceproductsfromopportunity.md)]

The GetInvoiceProductsFromOpportunity action does not return a value.

## Bound Entities 
Bound actions are invoked by appending the action name to the URI representing an entity or collection.
|EntityType|Binding Type|
|----------|-----------------|
|[invoice Entitytype](../entitytypes/invoice.md)|entity|

## Parameters 
Parameters allow for data to be passed to the action.
|Name|Type|Nullable|Unicode|Description|
|----------|-----------------|----------|-----------------|-----------------|  
|Opportunity|[opportunity EntityType](../entitytypes/opportunity.md)|false|true|The opportunity to retrieve the products from. |
