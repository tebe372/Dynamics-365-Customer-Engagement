# ResetUserFilters Action
[!INCLUDE[./descriptions/resetuserfilters.md](./descriptions/resetuserfilters.md)]

The ResetUserFilters action:
* Is not bound to any entity types.
* Does not return a value.



## Parameters 
Parameters allow for data to be passed to the action.
|Name|Type|Nullable|Unicode|Description|
|----------|-----------------|----------|-----------------|-----------------|  
|QueryType|Edm.Int32|false|true|The type of filters to set.|
