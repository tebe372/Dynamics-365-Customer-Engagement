# DeleteAndPromote Action
[!INCLUDE[./descriptions/deleteandpromote.md](./descriptions/deleteandpromote.md)]

The DeleteAndPromote action is not bound to any entity types.



## Parameters 
Parameters allow for data to be passed to the action.
|Name|Type|Nullable|Unicode|Description|
|----------|-----------------|----------|-----------------|-----------------|  
|UniqueName|Edm.String|false|false|The unique name of the solution being replaced.|

## Return Type
The DeleteAndPromote action returns the following value.
|Type|Nullable|Description| 
|----------|-----------------|-----------------|    
|[DeleteAndPromoteResponse ComplexType](../complextypes/DeleteAndPromoteResponse.md)|false|Contains the response from the DeleteAndPromote Action.|
