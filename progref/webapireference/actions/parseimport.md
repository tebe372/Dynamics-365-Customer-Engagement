# ParseImport Action
[!INCLUDE[./descriptions/parseimport.md](./descriptions/parseimport.md)]

The ParseImport action does not require any parameters.

## Bound Entities 
Bound actions are invoked by appending the action name to the URI representing an entity or collection.
|EntityType|Binding Type|
|----------|-----------------|
|[import Entitytype](../entitytypes/import.md)|entity|



## Return Type
The ParseImport action returns the following value.
|Type|Nullable|Description| 
|----------|-----------------|-----------------|    
|[asyncoperation EntityType](../entitytypes/asyncoperation.md)|false|The asynchronous job that parses the import files for this import.|
