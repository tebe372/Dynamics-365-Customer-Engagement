# msdyn_RetrieveDistanceMatrix Action
[!INCLUDE[./descriptions/msdyn_retrievedistancematrix.md](./descriptions/msdyn_retrievedistancematrix.md)]

The msdyn_RetrieveDistanceMatrix action is not bound to any entity types.



## Parameters 
Parameters allow for data to be passed to the action.
|Name|Type|Nullable|Unicode|Description|
|----------|-----------------|----------|-----------------|-----------------|  
|Sources|Collection([crmbaseentity EntityType](../entitytypes/crmbaseentity.md))|false|true||
|Targets|Collection([crmbaseentity EntityType](../entitytypes/crmbaseentity.md))|false|true||

## Return Type
The msdyn_RetrieveDistanceMatrix action returns the following value.
|Type|Nullable|Description| 
|----------|-----------------|-----------------|    
|Collection([crmbaseentity EntityType](../entitytypes/crmbaseentity.md))|false||
