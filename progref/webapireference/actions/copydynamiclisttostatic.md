# CopyDynamicListToStatic Action
[!INCLUDE[./descriptions/copydynamiclisttostatic.md](./descriptions/copydynamiclisttostatic.md)]

The CopyDynamicListToStatic action does not require any parameters.

## Bound Entities 
Bound actions are invoked by appending the action name to the URI representing an entity or collection.
|EntityType|Binding Type|
|----------|-----------------|
|[list Entitytype](../entitytypes/list.md)|entity|



## Return Type
The CopyDynamicListToStatic action returns the following value.
|Type|Nullable|Description| 
|----------|-----------------|-----------------|    
|[list EntityType](../entitytypes/list.md)|false|Contains the response from CopyDynamicListToStatic action.|
