# RemoveParent Action
[!INCLUDE[./descriptions/removeparent.md](./descriptions/removeparent.md)]

The RemoveParent action:
* Is not bound to any entity types.
* Does not return a value.



## Parameters 
Parameters allow for data to be passed to the action.
|Name|Type|Nullable|Unicode|Description|
|----------|-----------------|----------|-----------------|-----------------|  
|Target|[crmbaseentity EntityType](../entitytypes/crmbaseentity.md)|false|true|The target systemuser (user) record for the operation. |



## Entities
Use the RemoveParent action with this entity type:
|EntityType |Description| 
|----------|-----------------|    
|[systemuser Entitytype](../entitytypes/systemuser.md)|[!INCLUDE[../entitytypes/descriptions/systemuser.md](../entitytypes/descriptions/systemuser.md)]|
