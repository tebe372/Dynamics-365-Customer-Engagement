# CloneProduct Action
[!INCLUDE[./descriptions/cloneproduct.md](./descriptions/cloneproduct.md)]

The CloneProduct action does not require any parameters.

## Bound Entities 
Bound actions are invoked by appending the action name to the URI representing an entity or collection.
|EntityType|Binding Type|
|----------|-----------------|
|[product Entitytype](../entitytypes/product.md)|entity|



## Return Type
The CloneProduct action returns the following value.
|Type|Nullable|Description| 
|----------|-----------------|-----------------|    
|[product EntityType](../entitytypes/product.md)|false|Contains the response from CloneProduct action. |
