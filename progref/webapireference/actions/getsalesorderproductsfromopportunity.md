# GetSalesOrderProductsFromOpportunity Action
[!INCLUDE[./descriptions/getsalesorderproductsfromopportunity.md](./descriptions/getsalesorderproductsfromopportunity.md)]

The GetSalesOrderProductsFromOpportunity action does not return a value.

## Bound Entities 
Bound actions are invoked by appending the action name to the URI representing an entity or collection.
|EntityType|Binding Type|
|----------|-----------------|
|[salesorder Entitytype](../entitytypes/salesorder.md)|entity|

## Parameters 
Parameters allow for data to be passed to the action.
|Name|Type|Nullable|Unicode|Description|
|----------|-----------------|----------|-----------------|-----------------|  
|Opportunity|[opportunity EntityType](../entitytypes/opportunity.md)|false|true|The opportunity from which the products are to be retrieved.|
