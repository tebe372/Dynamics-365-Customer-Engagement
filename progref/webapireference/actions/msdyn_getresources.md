# msdyn_GetResources Action
[!INCLUDE[./descriptions/msdyn_getresources.md](./descriptions/msdyn_getresources.md)]

The msdyn_GetResources action is not bound to any entity types.



## Parameters 
Parameters allow for data to be passed to the action.
|Name|Type|Nullable|Unicode|Description|
|----------|-----------------|----------|-----------------|-----------------|  
|Page|Edm.Int32|false|true||
|PageSize|Edm.Int32|false|true||
|QueryString|Edm.String|false|false||

## Return Type
The msdyn_GetResources action returns the following value.
|Type|Nullable|Description| 
|----------|-----------------|-----------------|    
|[msdyn_GetResourcesResponse ComplexType](../complextypes/msdyn_GetResourcesResponse.md)|false|msdyn_GetResourcesResponse|
