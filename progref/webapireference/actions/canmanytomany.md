# CanManyToMany Action
[!INCLUDE[./descriptions/canmanytomany.md](./descriptions/canmanytomany.md)]

The CanManyToMany action is not bound to any entity types.



## Parameters 
Parameters allow for data to be passed to the action.
|Name|Type|Nullable|Unicode|Description|
|----------|-----------------|----------|-----------------|-----------------|  
|EntityName|Edm.String|false|false|Name of the entity. |

## Return Type
The CanManyToMany action returns the following value.
|Type|Nullable|Description| 
|----------|-----------------|-----------------|    
|[CanManyToManyResponse ComplexType](../complextypes/CanManyToManyResponse.md)|false|Contains the response from CanManyToMany action. |
