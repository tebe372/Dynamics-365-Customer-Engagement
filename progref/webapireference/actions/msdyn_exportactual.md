# msdyn_ExportActual Action
[!INCLUDE[./descriptions/msdyn_exportactual.md](./descriptions/msdyn_exportactual.md)]

The msdyn_ExportActual action:
* Is not bound to any entity types.
* Does not require any parameters.
* Does not return a value.
