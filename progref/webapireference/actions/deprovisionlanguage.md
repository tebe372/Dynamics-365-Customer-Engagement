# DeprovisionLanguage Action
[!INCLUDE[./descriptions/deprovisionlanguage.md](./descriptions/deprovisionlanguage.md)]

The DeprovisionLanguage action:
* Is not bound to any entity types.
* Does not return a value.



## Parameters 
Parameters allow for data to be passed to the action.
|Name|Type|Nullable|Unicode|Description|
|----------|-----------------|----------|-----------------|-----------------|  
|Language|Edm.Int32|false|true|The language to deprovision. |
