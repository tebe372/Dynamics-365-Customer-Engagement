# CopyMembersList Action
[!INCLUDE[./descriptions/copymemberslist.md](./descriptions/copymemberslist.md)]

The CopyMembersList action does not return a value.

## Bound Entities 
Bound actions are invoked by appending the action name to the URI representing an entity or collection.
|EntityType|Binding Type|
|----------|-----------------|
|[list Entitytype](../entitytypes/list.md)|entity|

## Parameters 
Parameters allow for data to be passed to the action.
|Name|Type|Nullable|Unicode|Description|
|----------|-----------------|----------|-----------------|-----------------|  
|TargetList|[list EntityType](../entitytypes/list.md)|false|true|The ID of the target list.|
