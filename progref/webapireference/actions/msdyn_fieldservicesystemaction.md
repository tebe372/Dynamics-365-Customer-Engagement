# msdyn_FieldServiceSystemAction Action
[!INCLUDE[./descriptions/msdyn_fieldservicesystemaction.md](./descriptions/msdyn_fieldservicesystemaction.md)]

The msdyn_FieldServiceSystemAction action is not bound to any entity types.



## Parameters 
Parameters allow for data to be passed to the action.
|Name|Type|Nullable|Unicode|Description|
|----------|-----------------|----------|-----------------|-----------------|  
|Type|Edm.Int32|false|true||
|InputParameter|Edm.String|true|false||

## Return Type
The msdyn_FieldServiceSystemAction action returns the following value.
|Type|Nullable|Description| 
|----------|-----------------|-----------------|    
|[msdyn_FieldServiceSystemActionResponse ComplexType](../complextypes/msdyn_FieldServiceSystemActionResponse.md)|false||
