# DeleteAuditData Action
[!INCLUDE[./descriptions/deleteauditdata.md](./descriptions/deleteauditdata.md)]

The DeleteAuditData action is not bound to any entity types.



## Parameters 
Parameters allow for data to be passed to the action.
|Name|Type|Nullable|Unicode|Description|
|----------|-----------------|----------|-----------------|-----------------|  
|EndDate|Edm.DateTimeOffset|false|true|The end date and time. |

## Return Type
The DeleteAuditData action returns the following value.
|Type|Nullable|Description| 
|----------|-----------------|-----------------|    
|[DeleteAuditDataResponse ComplexType](../complextypes/DeleteAuditDataResponse.md)|false|Contains the response from the DeleteAuditData Action. |
