# CanBeReferencing Action
[!INCLUDE[./descriptions/canbereferencing.md](./descriptions/canbereferencing.md)]

The CanBeReferencing action is not bound to any entity types.



## Parameters 
Parameters allow for data to be passed to the action.
|Name|Type|Nullable|Unicode|Description|
|----------|-----------------|----------|-----------------|-----------------|  
|EntityName|Edm.String|false|false|Logical entity name.|

## Return Type
The CanBeReferencing action returns the following value.
|Type|Nullable|Description| 
|----------|-----------------|-----------------|    
|[CanBeReferencingResponse ComplexType](../complextypes/CanBeReferencingResponse.md)|false|Contains the response from CanBeReferencing action. |
