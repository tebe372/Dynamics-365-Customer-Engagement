# ExportTranslation Action
[!INCLUDE[./descriptions/exporttranslation.md](./descriptions/exporttranslation.md)]

The ExportTranslation action is not bound to any entity types.



## Parameters 
Parameters allow for data to be passed to the action.
|Name|Type|Nullable|Unicode|Description|
|----------|-----------------|----------|-----------------|-----------------|  
|SolutionName|Edm.String|false|false|Unique name of the solution.|

## Return Type
The ExportTranslation action returns the following value.
|Type|Nullable|Description| 
|----------|-----------------|-----------------|    
|[ExportTranslationResponse ComplexType](../complextypes/ExportTranslationResponse.md)|false|Contains the response from the ExportTranslation Action.|
