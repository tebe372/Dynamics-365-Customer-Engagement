# PublishDuplicateRule Action
[!INCLUDE[./descriptions/publishduplicaterule.md](./descriptions/publishduplicaterule.md)]

The PublishDuplicateRule action does not require any parameters.

## Bound Entities 
Bound actions are invoked by appending the action name to the URI representing an entity or collection.
|EntityType|Binding Type|
|----------|-----------------|
|[duplicaterule Entitytype](../entitytypes/duplicaterule.md)|entity|



## Return Type
The PublishDuplicateRule action returns the following value.
|Type|Nullable|Description| 
|----------|-----------------|-----------------|    
|[asyncoperation EntityType](../entitytypes/asyncoperation.md)|false|The asynchronous job for publishing a duplicate detection rule. |
