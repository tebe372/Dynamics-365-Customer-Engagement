# SetDataEncryptionKey Action
[!INCLUDE[./descriptions/setdataencryptionkey.md](./descriptions/setdataencryptionkey.md)]

The SetDataEncryptionKey action:
* Is not bound to any entity types.
* Does not return a value.



## Parameters 
Parameters allow for data to be passed to the action.
|Name|Type|Nullable|Unicode|Description|
|----------|-----------------|----------|-----------------|-----------------|  
|EncryptionKey|Edm.String|false|false|The value of the data encryption key. |
|ChangeEncryptionKey|Edm.Boolean|false|true|The operation to perform with the data encryption key |
