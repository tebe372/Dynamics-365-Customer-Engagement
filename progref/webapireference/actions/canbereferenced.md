# CanBeReferenced Action
[!INCLUDE[./descriptions/canbereferenced.md](./descriptions/canbereferenced.md)]

The CanBeReferenced action is not bound to any entity types.



## Parameters 
Parameters allow for data to be passed to the action.
|Name|Type|Nullable|Unicode|Description|
|----------|-----------------|----------|-----------------|-----------------|  
|EntityName|Edm.String|false|false|Logical entity name.|

## Return Type
The CanBeReferenced action returns the following value.
|Type|Nullable|Description| 
|----------|-----------------|-----------------|    
|[CanBeReferencedResponse ComplexType](../complextypes/CanBeReferencedResponse.md)|false|Contains the response from CanBeReferenced action. |
