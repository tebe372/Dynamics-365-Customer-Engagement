# RevertProduct Action
[!INCLUDE[./descriptions/revertproduct.md](./descriptions/revertproduct.md)]

The RevertProduct action:
* Is not bound to any entity types.
* Does not return a value.



## Parameters 
Parameters allow for data to be passed to the action.
|Name|Type|Nullable|Unicode|Description|
|----------|-----------------|----------|-----------------|-----------------|  
|Target|[crmbaseentity EntityType](../entitytypes/crmbaseentity.md)|false|true|The target product family, product, or bundle record for which you want to revert the changes done to its properties since it was last published. |
