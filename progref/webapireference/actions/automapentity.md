# AutoMapEntity Action
[!INCLUDE[./descriptions/automapentity.md](./descriptions/automapentity.md)]

The AutoMapEntity action:
* Is not bound to any entity types.
* Does not return a value.



## Parameters 
Parameters allow for data to be passed to the action.
|Name|Type|Nullable|Unicode|Description|
|----------|-----------------|----------|-----------------|-----------------|  
|EntityMapId|Edm.Guid|false|true|The entity map to overwrite when the automated mapping is performed.|
