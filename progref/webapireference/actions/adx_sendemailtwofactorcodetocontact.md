# adx_SendEmailTwoFactorCodeToContact Action
[!INCLUDE[./descriptions/adx_sendemailtwofactorcodetocontact.md](./descriptions/adx_sendemailtwofactorcodetocontact.md)]

This action is added by the [Dynamics 365 Portals - Identity Workflows Solution](../solutions/microsoftidentityworkflows.md).

The adx_SendEmailTwoFactorCodeToContact action does not return a value.

## Bound Entities 
Bound actions are invoked by appending the action name to the URI representing an entity or collection.
