# CopyCampaignResponse Action
[!INCLUDE[./descriptions/copycampaignresponse.md](./descriptions/copycampaignresponse.md)]

The CopyCampaignResponse action does not require any parameters.

## Bound Entities 
Bound actions are invoked by appending the action name to the URI representing an entity or collection.
|EntityType|Binding Type|
|----------|-----------------|
|[campaignresponse Entitytype](../entitytypes/campaignresponse.md)|entity|



## Return Type
The CopyCampaignResponse action returns the following value.
|Type|Nullable|Description| 
|----------|-----------------|-----------------|    
|[campaignresponse EntityType](../entitytypes/campaignresponse.md)|false|Contains the response from the CopyCampaignResponse action.|
