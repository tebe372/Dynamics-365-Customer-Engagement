# UnlockInvoicePricing Action
[!INCLUDE[./descriptions/unlockinvoicepricing.md](./descriptions/unlockinvoicepricing.md)]

The UnlockInvoicePricing action:
* Is not bound to any entity types.
* Does not return a value.



## Parameters 
Parameters allow for data to be passed to the action.
|Name|Type|Nullable|Unicode|Description|
|----------|-----------------|----------|-----------------|-----------------|  
|InvoiceId|Edm.Guid|false|true|ID of the invoice. |
