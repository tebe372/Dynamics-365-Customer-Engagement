# adx_SendPasswordResetToContact Action
[!INCLUDE[./descriptions/adx_sendpasswordresettocontact.md](./descriptions/adx_sendpasswordresettocontact.md)]

This action is added by the [Dynamics 365 Portals - Identity Workflows Solution](../solutions/microsoftidentityworkflows.md).

The adx_SendPasswordResetToContact action does not return a value.

## Bound Entities 
Bound actions are invoked by appending the action name to the URI representing an entity or collection.

|----------|-----------------|
|[contact Entitytype](../entitytypes/contact.md)|entity|
