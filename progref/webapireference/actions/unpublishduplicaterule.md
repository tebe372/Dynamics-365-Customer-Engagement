# UnpublishDuplicateRule Action
[!INCLUDE[./descriptions/unpublishduplicaterule.md](./descriptions/unpublishduplicaterule.md)]

The UnpublishDuplicateRule action:
* Is not bound to any entity types.
* Does not return a value.



## Parameters 
Parameters allow for data to be passed to the action.
|Name|Type|Nullable|Unicode|Description|
|----------|-----------------|----------|-----------------|-----------------|  
|DuplicateRuleId|Edm.Guid|false|true|the ID of the duplicate rule to be unpublished. |
