# UninstallSampleData Action
[!INCLUDE[./descriptions/uninstallsampledata.md](./descriptions/uninstallsampledata.md)]

The UninstallSampleData action:
* Is not bound to any entity types.
* Does not require any parameters.
* Does not return a value.
