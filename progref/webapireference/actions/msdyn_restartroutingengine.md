# msdyn_RestartRoutingEngine Action
[!INCLUDE[./descriptions/msdyn_restartroutingengine.md](./descriptions/msdyn_restartroutingengine.md)]

The msdyn_RestartRoutingEngine action:
* Is not bound to any entity types.
* Does not require any parameters.
* Does not return a value.
