# FetchXmlToQueryExpression Function
[!INCLUDE[./descriptions/fetchxmltoqueryexpression.md](./descriptions/fetchxmltoqueryexpression.md)]

The FetchXmlToQueryExpression function is not bound to any entity types.



## Parameters 
Parameters allow for data to be passed to the function.
|Name|Type|Nullable|Unicode|Description|
|----------|-----------------|----------|-----------------|-----------------|  
|FetchXml|Edm.String|false|false|The query to convert.|

## Return Type
The FetchXmlToQueryExpression function returns the following value.
|Type|Nullable|Description| 
|----------|-----------------|-----------------|    
|[FetchXmlToQueryExpressionResponse ComplexType](../complextypes/FetchXmlToQueryExpressionResponse.md)|false|Contains the response from the FetchXmlToQueryExpression Function.|
