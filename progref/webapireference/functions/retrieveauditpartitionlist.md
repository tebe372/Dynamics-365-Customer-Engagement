# RetrieveAuditPartitionList Function
[!INCLUDE[./descriptions/retrieveauditpartitionlist.md](./descriptions/retrieveauditpartitionlist.md)]

The RetrieveAuditPartitionList action:
* Is not bound to any entity types.
* Does not require any parameters.





## Return Type
The RetrieveAuditPartitionList function returns the following value.
|Type|Nullable|Description| 
|----------|-----------------|-----------------|    
|[RetrieveAuditPartitionListResponse ComplexType](../complextypes/RetrieveAuditPartitionListResponse.md)|false|Contains the results from the RetrieveAuditPartitionList function.|
