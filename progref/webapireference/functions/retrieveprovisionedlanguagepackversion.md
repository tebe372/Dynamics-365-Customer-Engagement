# RetrieveProvisionedLanguagePackVersion Function
[!INCLUDE[./descriptions/retrieveprovisionedlanguagepackversion.md](./descriptions/retrieveprovisionedlanguagepackversion.md)]

The RetrieveProvisionedLanguagePackVersion function is not bound to any entity types.



## Parameters 
Parameters allow for data to be passed to the function.
|Name|Type|Nullable|Unicode|Description|
|----------|-----------------|----------|-----------------|-----------------|  
|Language|Edm.Int32|false|true|The locale Id for the language. |

## Return Type
The RetrieveProvisionedLanguagePackVersion function returns the following value.
|Type|Nullable|Description| 
|----------|-----------------|-----------------|    
|[RetrieveProvisionedLanguagePackVersionResponse ComplexType](../complextypes/RetrieveProvisionedLanguagePackVersionResponse.md)|false|Contains the results from the RetrieveProvisionedLanguagePackVersion function. |
