# RetrieveAbsoluteAndSiteCollectionUrl Function
[!INCLUDE[./descriptions/retrieveabsoluteandsitecollectionurl.md](./descriptions/retrieveabsoluteandsitecollectionurl.md)]

The RetrieveAbsoluteAndSiteCollectionUrl function does not require any parameters.

## Bound Entities 
Bound functions are invoked by appending the function name to the URI representing an entity or collection.
|EntityType|Binding Type|
|----------|-----------------|
|[sharepointsite Entitytype](../entitytypes/sharepointsite.md)|entity|
|[sharepointdocumentlocation Entitytype](../entitytypes/sharepointdocumentlocation.md)|entity|



## Return Type
The RetrieveAbsoluteAndSiteCollectionUrl function returns the following value.
|Type|Nullable|Description| 
|----------|-----------------|-----------------|    
|[RetrieveAbsoluteAndSiteCollectionUrlResponse ComplexType](../complextypes/RetrieveAbsoluteAndSiteCollectionUrlResponse.md)|false|The response from this function|
