# CalculateTotalTimeIncident Function
[!INCLUDE[./descriptions/calculatetotaltimeincident.md](./descriptions/calculatetotaltimeincident.md)]

The CalculateTotalTimeIncident function does not require any parameters.

## Bound Entities 
Bound functions are invoked by appending the function name to the URI representing an entity or collection.
|EntityType|Binding Type|
|----------|-----------------|
|[incident Entitytype](../entitytypes/incident.md)|entity|



## Return Type
The CalculateTotalTimeIncident function returns the following value.
|Type|Nullable|Description| 
|----------|-----------------|-----------------|    
|[CalculateTotalTimeIncidentResponse ComplexType](../complextypes/CalculateTotalTimeIncidentResponse.md)|false|Contains the response from the CaluculateTotalTimeIncident function. |
