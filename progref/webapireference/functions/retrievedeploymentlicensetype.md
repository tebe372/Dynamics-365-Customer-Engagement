# RetrieveDeploymentLicenseType Function
[!INCLUDE[./descriptions/retrievedeploymentlicensetype.md](./descriptions/retrievedeploymentlicensetype.md)]

The RetrieveDeploymentLicenseType action:
* Is not bound to any entity types.
* Does not require any parameters.





## Return Type
The RetrieveDeploymentLicenseType function returns the following value.
|Type|Nullable|Description| 
|----------|-----------------|-----------------|    
|[RetrieveDeploymentLicenseTypeResponse ComplexType](../complextypes/RetrieveDeploymentLicenseTypeResponse.md)|false|Contains the results from the RetrieveDeploymentLicenseType function.|
