# RetrieveMissingComponents Function
[!INCLUDE[./descriptions/retrievemissingcomponents.md](./descriptions/retrievemissingcomponents.md)]

The RetrieveMissingComponents function is not bound to any entity types.



## Parameters 
Parameters allow for data to be passed to the function.
|Name|Type|Nullable|Unicode|Description|
|----------|-----------------|----------|-----------------|-----------------|  
|CustomizationFile|Edm.Binary|false|true|A file for the solution. |

## Return Type
The RetrieveMissingComponents function returns the following value.
|Type|Nullable|Description| 
|----------|-----------------|-----------------|    
|[RetrieveMissingComponentsResponse ComplexType](../complextypes/RetrieveMissingComponentsResponse.md)|false|Contains the results from the  RetrieveMissingComponents function. |
