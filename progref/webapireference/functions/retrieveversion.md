# RetrieveVersion Function
[!INCLUDE[./descriptions/retrieveversion.md](./descriptions/retrieveversion.md)]

The RetrieveVersion action:
* Is not bound to any entity types.
* Does not require any parameters.





## Return Type
The RetrieveVersion function returns the following value.
|Type|Nullable|Description| 
|----------|-----------------|-----------------|    
|[RetrieveVersionResponse ComplexType](../complextypes/RetrieveVersionResponse.md)|false|Contains the results from the RetrieveVersion function |
