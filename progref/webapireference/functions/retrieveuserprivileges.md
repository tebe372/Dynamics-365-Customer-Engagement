# RetrieveUserPrivileges Function
[!INCLUDE[./descriptions/retrieveuserprivileges.md](./descriptions/retrieveuserprivileges.md)]

The RetrieveUserPrivileges function does not require any parameters.

## Bound Entities 
Bound functions are invoked by appending the function name to the URI representing an entity or collection.
|EntityType|Binding Type|
|----------|-----------------|
|[systemuser Entitytype](../entitytypes/systemuser.md)|entity|



## Return Type
The RetrieveUserPrivileges function returns the following value.
|Type|Nullable|Description| 
|----------|-----------------|-----------------|    
|[RetrieveUserPrivilegesResponse ComplexType](../complextypes/RetrieveUserPrivilegesResponse.md)|false|Contains the results from the RetrieveUserPrivilegesResponse function. |
