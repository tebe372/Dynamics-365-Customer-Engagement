# RetrieveApplicationRibbon Function
[!INCLUDE[./descriptions/retrieveapplicationribbon.md](./descriptions/retrieveapplicationribbon.md)]

The RetrieveApplicationRibbon action:
* Is not bound to any entity types.
* Does not require any parameters.





## Return Type
The RetrieveApplicationRibbon function returns the following value.
|Type|Nullable|Description| 
|----------|-----------------|-----------------|    
|[RetrieveApplicationRibbonResponse ComplexType](../complextypes/RetrieveApplicationRibbonResponse.md)|false|Contains the results from the RetrieveApplicationRibbon function.|
