# GetValidManyToMany Function
[!INCLUDE[./descriptions/getvalidmanytomany.md](./descriptions/getvalidmanytomany.md)]

The GetValidManyToMany action:
* Is not bound to any entity types.
* Does not require any parameters.





## Return Type
The GetValidManyToMany function returns the following value.
|Type|Nullable|Description| 
|----------|-----------------|-----------------|    
|[GetValidManyToManyResponse ComplexType](../complextypes/GetValidManyToManyResponse.md)|false|Contains the results from the GetValidManyToMany function.|
