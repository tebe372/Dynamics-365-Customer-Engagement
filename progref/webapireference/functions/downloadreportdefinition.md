# DownloadReportDefinition Function
[!INCLUDE[./descriptions/downloadreportdefinition.md](./descriptions/downloadreportdefinition.md)]

The DownloadReportDefinition function does not require any parameters.

## Bound Entities 
Bound functions are invoked by appending the function name to the URI representing an entity or collection.
|EntityType|Binding Type|
|----------|-----------------|
|[report Entitytype](../entitytypes/report.md)|entity|



## Return Type
The DownloadReportDefinition function returns the following value.
|Type|Nullable|Description| 
|----------|-----------------|-----------------|    
|[DownloadReportDefinitionResponse ComplexType](../complextypes/DownloadReportDefinitionResponse.md)|false|Contains the response from DownloadReportDefinition function |
