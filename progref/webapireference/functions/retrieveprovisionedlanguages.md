# RetrieveProvisionedLanguages Function
[!INCLUDE[./descriptions/retrieveprovisionedlanguages.md](./descriptions/retrieveprovisionedlanguages.md)]

The RetrieveProvisionedLanguages action:
* Is not bound to any entity types.
* Does not require any parameters.





## Return Type
The RetrieveProvisionedLanguages function returns the following value.
|Type|Nullable|Description| 
|----------|-----------------|-----------------|    
|[RetrieveProvisionedLanguagesResponse ComplexType](../complextypes/RetrieveProvisionedLanguagesResponse.md)|false|Contains the results from the RetrieveProvisionedLanguages function. |
