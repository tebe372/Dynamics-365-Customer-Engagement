# RetrieveDeprovisionedLanguages Function
[!INCLUDE[./descriptions/retrievedeprovisionedlanguages.md](./descriptions/retrievedeprovisionedlanguages.md)]

The RetrieveDeprovisionedLanguages action:
* Is not bound to any entity types.
* Does not require any parameters.





## Return Type
The RetrieveDeprovisionedLanguages function returns the following value.
|Type|Nullable|Description| 
|----------|-----------------|-----------------|    
|[RetrieveDeprovisionedLanguagesResponse ComplexType](../complextypes/RetrieveDeprovisionedLanguagesResponse.md)|false|Contains the results from the RetrieveDeprovisionedLanguages function. |
