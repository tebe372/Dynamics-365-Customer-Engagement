# RetrieveSubGroupsResourceGroup Function
[!INCLUDE[./descriptions/retrievesubgroupsresourcegroup.md](./descriptions/retrievesubgroupsresourcegroup.md)]

The RetrieveSubGroupsResourceGroup function does not require any parameters.

## Bound Entities 
Bound functions are invoked by appending the function name to the URI representing an entity or collection.
|EntityType|Binding Type|
|----------|-----------------|
|[resourcegroup Entitytype](../entitytypes/resourcegroup.md)|entity|



## Return Type
The RetrieveSubGroupsResourceGroup function returns the following value.
|Type|Nullable|Description| 
|----------|-----------------|-----------------|    
|Collection([resourcegroup EntityType](../entitytypes/resourcegroup.md))|false|Contains the results from the RetrieveSubGroupsResourceGroup function. |
