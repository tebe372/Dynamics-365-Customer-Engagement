# RetrieveInstalledLanguagePacks Function
[!INCLUDE[./descriptions/retrieveinstalledlanguagepacks.md](./descriptions/retrieveinstalledlanguagepacks.md)]

The RetrieveInstalledLanguagePacks action:
* Is not bound to any entity types.
* Does not require any parameters.





## Return Type
The RetrieveInstalledLanguagePacks function returns the following value.
|Type|Nullable|Description| 
|----------|-----------------|-----------------|    
|[RetrieveInstalledLanguagePacksResponse ComplexType](../complextypes/RetrieveInstalledLanguagePacksResponse.md)|false|Contains the results from the RetrieveInstalledLanguagePacks function. |
