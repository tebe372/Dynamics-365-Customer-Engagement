# RetrievePrincipalSyncAttributeMappings Function
[!INCLUDE[./descriptions/retrieveprincipalsyncattributemappings.md](./descriptions/retrieveprincipalsyncattributemappings.md)]

The RetrievePrincipalSyncAttributeMappings function does not require any parameters.

## Bound Entities 
Bound functions are invoked by appending the function name to the URI representing an entity or collection.
|EntityType|Binding Type|
|----------|-----------------|
|[systemuser Entitytype](../entitytypes/systemuser.md)|entity|
|[team Entitytype](../entitytypes/team.md)|entity|



## Return Type
The RetrievePrincipalSyncAttributeMappings function returns the following value.
|Type|Nullable|Description| 
|----------|-----------------|-----------------|    
|[RetrievePrincipalSyncAttributeMappingsResponse ComplexType](../complextypes/RetrievePrincipalSyncAttributeMappingsResponse.md)|false|For internal use only.|
