# WhoAmI Function
[!INCLUDE[./descriptions/whoami.md](./descriptions/whoami.md)]

The WhoAmI action:
* Is not bound to any entity types.
* Does not require any parameters.





## Return Type
The WhoAmI function returns the following value.
|Type|Nullable|Description| 
|----------|-----------------|-----------------|    
|[WhoAmIResponse ComplexType](../complextypes/WhoAmIResponse.md)|false|Contains the response from the WhoAmI function. |
