# RetrieveInstalledLanguagePackVersion Function
[!INCLUDE[./descriptions/retrieveinstalledlanguagepackversion.md](./descriptions/retrieveinstalledlanguagepackversion.md)]

The RetrieveInstalledLanguagePackVersion function is not bound to any entity types.



## Parameters 
Parameters allow for data to be passed to the function.
|Name|Type|Nullable|Unicode|Description|
|----------|-----------------|----------|-----------------|-----------------|  
|Language|Edm.Int32|false|true|The installed language pack. |

## Return Type
The RetrieveInstalledLanguagePackVersion function returns the following value.
|Type|Nullable|Description| 
|----------|-----------------|-----------------|    
|[RetrieveInstalledLanguagePackVersionResponse ComplexType](../complextypes/RetrieveInstalledLanguagePackVersionResponse.md)|false|Contains the response from the RetrieveInstalledLanguagePackVersion function. |
