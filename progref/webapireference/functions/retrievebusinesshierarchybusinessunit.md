# RetrieveBusinessHierarchyBusinessUnit Function
[!INCLUDE[./descriptions/retrievebusinesshierarchybusinessunit.md](./descriptions/retrievebusinesshierarchybusinessunit.md)]

The RetrieveBusinessHierarchyBusinessUnit function does not require any parameters.

## Bound Entities 
Bound functions are invoked by appending the function name to the URI representing an entity or collection.
|EntityType|Binding Type|
|----------|-----------------|
|[businessunit Entitytype](../entitytypes/businessunit.md)|entity|



## Return Type
The RetrieveBusinessHierarchyBusinessUnit function returns the following value.
|Type|Nullable|Description| 
|----------|-----------------|-----------------|    
|Collection([businessunit EntityType](../entitytypes/businessunit.md))|false|Contains the results from the RetrieveBusinessHierarchyBusinessUnit function.|
