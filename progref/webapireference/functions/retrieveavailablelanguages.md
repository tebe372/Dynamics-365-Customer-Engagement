# RetrieveAvailableLanguages Function
[!INCLUDE[./descriptions/retrieveavailablelanguages.md](./descriptions/retrieveavailablelanguages.md)]

The RetrieveAvailableLanguages action:
* Is not bound to any entity types.
* Does not require any parameters.





## Return Type
The RetrieveAvailableLanguages function returns the following value.
|Type|Nullable|Description| 
|----------|-----------------|-----------------|    
|[RetrieveAvailableLanguagesResponse ComplexType](../complextypes/RetrieveAvailableLanguagesResponse.md)|false|Contains the results from the RetrieveAvailableLanguages function. |
