# RetrieveLicenseInfo Function
[!INCLUDE[./descriptions/retrievelicenseinfo.md](./descriptions/retrievelicenseinfo.md)]

The RetrieveLicenseInfo function is not bound to any entity types.



## Parameters 
Parameters allow for data to be passed to the function.
|Name|Type|Nullable|Unicode|Description|
|----------|-----------------|----------|-----------------|-----------------|  
|AccessMode|Edm.Int32|false|true|The access mode for retrieving the license information.|

## Return Type
The RetrieveLicenseInfo function returns the following value.
|Type|Nullable|Description| 
|----------|-----------------|-----------------|    
|[RetrieveLicenseInfoResponse ComplexType](../complextypes/RetrieveLicenseInfoResponse.md)|false|Contains the results from the RetrieveLicenseInfo function.|
