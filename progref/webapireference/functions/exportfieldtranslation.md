# ExportFieldTranslation Function
[!INCLUDE[./descriptions/exportfieldtranslation.md](./descriptions/exportfieldtranslation.md)]

The ExportFieldTranslation action:
* Is not bound to any entity types.
* Does not require any parameters.





## Return Type
The ExportFieldTranslation function returns the following value.
|Type|Nullable|Description| 
|----------|-----------------|-----------------|    
|[ExportFieldTranslationResponse ComplexType](../complextypes/ExportFieldTranslationResponse.md)|false|Contains the response from the ExportFieldTranslation function.|
