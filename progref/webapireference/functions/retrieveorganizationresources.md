# RetrieveOrganizationResources Function
[!INCLUDE[./descriptions/retrieveorganizationresources.md](./descriptions/retrieveorganizationresources.md)]

The RetrieveOrganizationResources action:
* Is not bound to any entity types.
* Does not require any parameters.





## Return Type
The RetrieveOrganizationResources function returns the following value.
|Type|Nullable|Description| 
|----------|-----------------|-----------------|    
|[RetrieveOrganizationResourcesResponse ComplexType](../complextypes/RetrieveOrganizationResourcesResponse.md)|false|Contains the results from the RetrieveOrganizationResources function. |
