# RetrievePrincipalAttributePrivileges Function
[!INCLUDE[./descriptions/retrieveprincipalattributeprivileges.md](./descriptions/retrieveprincipalattributeprivileges.md)]

The RetrievePrincipalAttributePrivileges function does not require any parameters.

## Bound Entities 
Bound functions are invoked by appending the function name to the URI representing an entity or collection.
|EntityType|Binding Type|
|----------|-----------------|
|[systemuser Entitytype](../entitytypes/systemuser.md)|entity|
|[team Entitytype](../entitytypes/team.md)|entity|



## Return Type
The RetrievePrincipalAttributePrivileges function returns the following value.
|Type|Nullable|Description| 
|----------|-----------------|-----------------|    
|[RetrievePrincipalAttributePrivilegesResponse ComplexType](../complextypes/RetrievePrincipalAttributePrivilegesResponse.md)|false|Contains the results of the RetrievePrincipalAttributePrivileges function. |
