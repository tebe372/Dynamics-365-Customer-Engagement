# RetrieveByResourceResourceGroup Function
[!INCLUDE[./descriptions/retrievebyresourceresourcegroup.md](./descriptions/retrievebyresourceresourcegroup.md)]

The RetrieveByResourceResourceGroup function does not require any parameters.

## Bound Entities 
Bound functions are invoked by appending the function name to the URI representing an entity or collection.
|EntityType|Binding Type|
|----------|-----------------|
|[resource Entitytype](../entitytypes/resource.md)|entity|



## Return Type
The RetrieveByResourceResourceGroup function returns the following value.
|Type|Nullable|Description| 
|----------|-----------------|-----------------|    
|Collection([resourcegroup EntityType](../entitytypes/resourcegroup.md))|false|Contains the results from the RetrieveByResourceResourceGroup function.|

|EntityType |Description| 
|----------|-----------------|
