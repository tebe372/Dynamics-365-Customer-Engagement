# IsDataEncryptionActive Function
[!INCLUDE[./descriptions/isdataencryptionactive.md](./descriptions/isdataencryptionactive.md)]

The IsDataEncryptionActive action:
* Is not bound to any entity types.
* Does not require any parameters.





## Return Type
The IsDataEncryptionActive function returns the following value.
|Type|Nullable|Description| 
|----------|-----------------|-----------------|    
|[IsDataEncryptionActiveResponse ComplexType](../complextypes/IsDataEncryptionActiveResponse.md)|false|Whether data encryption is currently running. |
