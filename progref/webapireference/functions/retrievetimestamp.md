# RetrieveTimestamp Function
[!INCLUDE[./descriptions/retrievetimestamp.md](./descriptions/retrievetimestamp.md)]

The RetrieveTimestamp action:
* Is not bound to any entity types.
* Does not require any parameters.





## Return Type
The RetrieveTimestamp function returns the following value.
|Type|Nullable|Description| 
|----------|-----------------|-----------------|    
|[RetrieveTimestampResponse ComplexType](../complextypes/RetrieveTimestampResponse.md)|false|Contains the results from the the RetrieveTimestamp function. |
