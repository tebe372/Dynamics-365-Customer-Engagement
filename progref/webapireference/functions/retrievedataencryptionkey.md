# RetrieveDataEncryptionKey Function
[!INCLUDE[./descriptions/retrievedataencryptionkey.md](./descriptions/retrievedataencryptionkey.md)]

The RetrieveDataEncryptionKey action:
* Is not bound to any entity types.
* Does not require any parameters.





## Return Type
The RetrieveDataEncryptionKey function returns the following value.
|Type|Nullable|Description| 
|----------|-----------------|-----------------|    
|[RetrieveDataEncryptionKeyResponse ComplexType](../complextypes/RetrieveDataEncryptionKeyResponse.md)|false|Contains the results from the RetrieveDataEncryptionKey function. |
