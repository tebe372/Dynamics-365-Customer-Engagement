# RetrievePrivilegeSet Function
[!INCLUDE[./descriptions/retrieveprivilegeset.md](./descriptions/retrieveprivilegeset.md)]

The RetrievePrivilegeSet function does not require any parameters.

## Bound Entities 
Bound functions are invoked by appending the function name to the URI representing an entity or collection.
|EntityType|Binding Type|
|----------|-----------------|
|[privilege Entitytype](../entitytypes/privilege.md)|collection|



## Return Type
The RetrievePrivilegeSet function returns the following value.
|Type|Nullable|Description| 
|----------|-----------------|-----------------|    
|Collection([privilege EntityType](../entitytypes/privilege.md))|false|A collection of privileges|
