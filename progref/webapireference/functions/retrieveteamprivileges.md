# RetrieveTeamPrivileges Function
[!INCLUDE[./descriptions/retrieveteamprivileges.md](./descriptions/retrieveteamprivileges.md)]

The RetrieveTeamPrivileges function does not require any parameters.

## Bound Entities 
Bound functions are invoked by appending the function name to the URI representing an entity or collection.
|EntityType|Binding Type|
|----------|-----------------|
|[team Entitytype](../entitytypes/team.md)|entity|



## Return Type
The RetrieveTeamPrivileges function returns the following value.
|Type|Nullable|Description| 
|----------|-----------------|-----------------|    
|[RetrieveTeamPrivilegesResponse ComplexType](../complextypes/RetrieveTeamPrivilegesResponse.md)|false|Contains the results from the RetrieveTeamPrivileges function. |
