# RetrieveMailboxTrackingFolders Function
[!INCLUDE[./descriptions/retrievemailboxtrackingfolders.md](./descriptions/retrievemailboxtrackingfolders.md)]

The RetrieveMailboxTrackingFolders function is not bound to any entity types.



## Parameters 
Parameters allow for data to be passed to the function.
|Name|Type|Nullable|Unicode|Description|
|----------|-----------------|----------|-----------------|-----------------|  
|MailboxId|Edm.String|false|false|The mailbox ID of the user for which you want to retrieve the folder-level tracking rules.|

## Return Type
The RetrieveMailboxTrackingFolders function returns the following value.
|Type|Nullable|Description| 
|----------|-----------------|-----------------|    
|[RetrieveMailboxTrackingFoldersResponse ComplexType](../complextypes/RetrieveMailboxTrackingFoldersResponse.md)|false|Contains the results from the RetrieveMailboxTrackingFolders function. |
