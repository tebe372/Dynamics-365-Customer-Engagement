# GetReportHistoryLimit Function
[!INCLUDE[./descriptions/getreporthistorylimit.md](./descriptions/getreporthistorylimit.md)]

The GetReportHistoryLimit function does not require any parameters.

## Bound Entities 
Bound functions are invoked by appending the function name to the URI representing an entity or collection.
|EntityType|Binding Type|
|----------|-----------------|
|[report Entitytype](../entitytypes/report.md)|entity|



## Return Type
The GetReportHistoryLimit function returns the following value.
|Type|Nullable|Description| 
|----------|-----------------|-----------------|    
|[GetReportHistoryLimitResponse ComplexType](../complextypes/GetReportHistoryLimitResponse.md)|false|Contains the results from the GetReportHistoryLimit function.|
