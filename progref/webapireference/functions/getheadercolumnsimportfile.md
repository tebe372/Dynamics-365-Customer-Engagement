# GetHeaderColumnsImportFile Function
[!INCLUDE[./descriptions/getheadercolumnsimportfile.md](./descriptions/getheadercolumnsimportfile.md)]

The GetHeaderColumnsImportFile function does not require any parameters.

## Bound Entities 
Bound functions are invoked by appending the function name to the URI representing an entity or collection.
|EntityType|Binding Type|
|----------|-----------------|
|[importfile Entitytype](../entitytypes/importfile.md)|entity|



## Return Type
The GetHeaderColumnsImportFile function returns the following value.
|Type|Nullable|Description| 
|----------|-----------------|-----------------|    
|[GetHeaderColumnsImportFileResponse ComplexType](../complextypes/GetHeaderColumnsImportFileResponse.md)|false|The response from this function.|
