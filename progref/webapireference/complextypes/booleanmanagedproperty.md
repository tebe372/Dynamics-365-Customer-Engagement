# BooleanManagedProperty ComplexType
[!INCLUDE[./descriptions/booleanmanagedproperty.md](./descriptions/booleanmanagedproperty.md)]



## Properties
The properties of a complex type contain the structured data of the type.
|Name|Type|Nullable|Unicode|Description| 
|----|----|--------|-------|-----------| 
|Value|Edm.Boolean|true|true|The value of the managed property.  | 
|CanBeChanged|Edm.Boolean|true|true|Whether the managed property value can be changed. | 
|ManagedPropertyLogicalName|Edm.String|true|false|The logical name for the managed property. |
