# StateOptionMetadata ComplexType
[!INCLUDE[./descriptions/stateoptionmetadata.md](./descriptions/stateoptionmetadata.md)]

This Web API complex type inherits from [OptionMetadata ComplexType](../complextypes/OptionMetadata.md).

## Properties
The properties of a complex type contain the structured data of the type.
|Name|Type|Nullable|Unicode|Description| 
|----|----|--------|-------|-----------| 
|DefaultStatus|Edm.Int32|true|true| The default status value associated with this state. | 
|InvariantName|Edm.String|true|false|The name of the state that does not change. |
