[!INCLUDE[./descriptions/canmanytomanyresponse.md](./descriptions/canmanytomanyresponse.md)]



## Properties
The properties of a complex type contain the structured data of the type.
|Name|Type|Nullable|Unicode|Description| 
|----|----|--------|-------|-----------| 
|CanManyToMany|Edm.Boolean|true|true|The result of the request to see whether the entity can participate in a many-to-many relationship.|
